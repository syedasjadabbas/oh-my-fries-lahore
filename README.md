## 🛠️ Getting Started

### Prerequisites

- **Node.js** `v18+`
- **npm** `v9+`

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/syedasjadabbas/oh-my-fries-lahore.git

# 2. Navigate into the project
cd oh-my-fries-lahore

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser. 🎉

## 🌟 Highlights

- **WhatsApp Integration** — Customers can place orders directly via WhatsApp with a pre-filled cart summary, no app download needed.
- **Animated UI** — Every card, modal, and page transition is enhanced with Framer Motion spring animations for a premium feel.
- **Type-Safe Codebase** — Strict TypeScript interfaces for `MenuItem` and `CartItem` ensure reliability throughout.
- **Oxlint** — Ultra-fast Rust-based linter keeping the codebase clean and consistent.
